Pollen Tracker is a Fitbit App that gives users Real Time Information about Allergens in the air.

How it works:
Accesses the User's latitude and longitude coordinates from the fitbit.
Makes a call to the geonames web API in order to retrieve the ZIP code the user is in
Uses the nasacort web API to get the current allergens in the air, along with the pollen forecast for the next 3 days.
Presents this information to the user
